# CognoRise-Infotech--Task-2

Currency Converter
The Currency Converter is a web application designed to help users convert amounts between different currencies. It leverages real-time exchange rate data from a reliable API, allowing users to input an amount and select a currency to see its equivalent values in various other currencies.

Features
Responsive Design: Utilizes Bootstrap for a clean, responsive layout that works seamlessly across different devices.
Real-Time Data: Fetches up-to-date exchange rates from the ExchangeRate API.
User-Friendly Interface: Simple and intuitive form for users to input amounts and select currencies.
Dynamic Data Presentation: Displays conversion results in a structured table format for easy viewing.
Technologies Used
HTML5: Provides the structure of the web page.
CSS3: Custom styles and Bootstrap for responsive design and layout.
JavaScript: Handles API calls and dynamically updates the conversion table.
Bootstrap 5.3.3: Ensures a responsive and modern design.
